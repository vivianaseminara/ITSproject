Overview

This project was developed by a group of students from the University of Bologna with the aim of conducting an in-depth analysis using cutting-edge technologies to extract information from knowledge graphs. It not only seeks to gather and organize information from various sources but also to enhance and make it more accessible using large language models (LLMs). The ability to extract meaningful information from knowledge graphs represents an emerging field of research with the potential to transform how we interpret and use data. This project aims to explore these technologies in an academic setting, contributing to the development of new methods and tools that can improve how information is managed and analyzed across different fields. The system obtained not only collects data but also integrates artificial intelligence. The large language models used in this project can understand and generate text in a highly sophisticated manner, thereby offering new opportunities for data enrichment and knowledge creation.
