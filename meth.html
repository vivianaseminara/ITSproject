SPARQL

In questa sezione vengono analizzate le query SPARQL utilizzate per interrogare il grafo di conoscenza. Per ciascuna query, vengono forniti i seguenti elementi:

  • La query vera e propria.
  • Un'immagine che mostra i risultati dell'interrogazione.
  • Una breve analisi dei risultati estrapolati, se presente.
Questa struttura permette una comprensione chiara e dettagliata delle interrogazioni effettuate e dei dati ottenuti.

1° Question:

What are the churches with name and description?

Query:


SELECT DISTINCT ?tipo
WHERE {
  ?bene a arco:ArchitecturalOrLandscapeHeritage ;
        dc:type ?tipo .
}

Result

Analysis

La query cerca tutti i beni architettonici o paesaggistici nel grafo di conoscenza e estrae i tipi unici di questi beni.

2° Question:

What are the churches with name and description?

Query:

SELECT ?sub ?dec
                    WHERE {
                      {
                        ?cp a arco:ArchitecturalOrLandscapeHeritage ;
                            dc:type ?o ;
                            dc:subject ?sub ;
                            dc:description ?dec .
                        FILTER(str(?o) = 'chiesa')
                      }
                      UNION
                      {
                        ?cp2 a arco:ArchitecturalOrLandscapeHeritage ;
                             dc:type ?o2 ;
                             dc:subject ?sub2 ;
                             dc:description ?dec2 .
                        FILTER(str(?o2) = 'cattedrale')
                      }
                    }

Result

Analysis

La query cerca tutti i beni architettonici o paesaggistici (rappresentati da ?cp e ?cp2) che sono di tipo 'chiesa' o cattedrale. Per ogni chiesa o cattedrale trovata, la query restituisce il soggetto (?sub) e la descrizione (?dec) associati. Osservazione: Le cattedrali presenti nel grafo sono limitate, approssimativamente 7. Pertanto, si è deciso di procedere esclusivamente con l'analisi delle chiese.

3° Question:

What are the churches with name and description and if present the author?

Query:

SELECT ?sub ?dec ?auth
                    WHERE {
                      ?cp a arco:ArchitecturalOrLandscapeHeritage ;
                          dc:type ?o ;
                          dc:subject ?sub ;
                          dc:description ?dec .
                      OPTIONAL { ?cp a-cd:hasAuthor ?auth } .
                      FILTER(str(?o) = 'chiesa')
                    }

Result

Analysis

La query cerca tutti i beni architettonici o paesaggistici (rappresentati da ?cp) che sono di tipo 'chiesa'. Per ogni chiesa trovata, la query restituisce il soggetto (?sub), la descrizione (?dec) e, se disponibile, l'autore (?auth).

4° Question:

How many churches are there?

Query:

SELECT (COUNT(?cp) AS ?numeroChiese)
                    WHERE {
                      ?cp a arco:ArchitecturalOrLandscapeHeritage ;
                          dc:type ?o .
                      FILTER(str(?o) = 'chiesa')
                    }

Result

Analysis

La query cerca tutti i beni architettonici o paesaggistici (rappresentati da ?cp) che sono di tipo 'chiesa' e conta quanti di questi beni esistono. Il risultato è il numero totale di chiese presenti nel grafo di conoscenza.

5° Question:

What is the number of churches grouped by region?

Query:

SELECT (COUNT(?cp) AS ?numeroChiese)
                    SELECT ?regionId (COUNT(?regionId) AS ?regionCount)
                        WHERE {
                        ?cp a arco:ArchitecturalOrLandscapeHeritage ;
                            dc:type ?o ;
                        arco:regionIdentifier ?regionId .
                        FILTER(str(?o) = 'chiesa')
                        }
                        GROUP BY ?regionId
                        ORDER BY DESC(?regionCount)

Result

Analysis

La query cerca tutti i beni architettonici o paesaggistici (rappresentati da ?cp) che sono di tipo 'chiesa' e sono associati a una regione (?regionId). Poi conta il numero di chiese per ciascuna regione e ordina i risultati in modo decrescente secondo il numero di chiese. Da questa query possiamo notare una distribuzione non omogenea del numero di chiese presenti nelle diverse regioni. In particolare, analizzeremo successivamente le chiese e gli stili architettonici presenti in Sicilia, che conta solo 26 chiese identificate dall'ID 19. Questo dato evidenzia alcune lacune da colmare all'interno del Knowledge Graph.

6° Question:

What are all the styles concerning barrocco in italy?

Query:


                    PREFIX dcterms: 

                    SELECT ?lStile (COUNT(?stile) AS ?stileCount)
                    WHERE {
                    ?cp a arco:ArchitecturalOrLandscapeHeritage ;
                        dc:type ?o ;
                        a-cd:hasAuthorshipAttribution ?Auth ;
                        dcterms:spatial ?loc .
                    ?loc rdfs:label ?l .
                    ?Auth a-cd:hasCulturalScope ?stile .
                    ?stile rdfs:label ?lStile .
                    FILTER(str(?o) = "chiesa") 
                    FILTER(REGEX(?lStile, "barocco", "i"))
                    }
                    GROUP BY ?lStile
                    ORDER BY DESC(?stileCount)

Result

Analysis

La query cerca tutti i beni architettonici o paesaggistici di tipo "chiesa" e con stile culturale contenente la parola "barocco". Conta il numero di occorrenze di ciascuno stile barocco e ordina i risultati in ordine decrescente secondo il numero di occorrenze. Ora concentriamo la nostra attenzione sugli stili architettonici presenti in Italia, in particolare sullo stile barocco. Dalle fonti disponibili online, emerge che lo stile barocco è prevalentemente presente nelle regioni Sicilia, Lazio (con Roma) e Puglia. Vogliamo verificare questa informazione interrogando il Knowledge Graph (KG) per confermarla.

7° Question:

Qualsi sono le chiese siciliane?

Query:


                    PREFIX dcterms: 

                    SELECT ?cp ?loc ?l ?lStile
                    WHERE {
                      ?cp a arco:ArchitecturalOrLandscapeHeritage ;
                          dc:type ?o ;
                          a-cd:hasAuthorshipAttribution ?Auth ;
                          dcterms:spatial ?loc .
                      ?loc rdfs:label ?l .
                      ?Auth a-cd:hasCulturalScope ?stile .
                    ?stile rdfs:label ?lStile .
                      FILTER(str(?o) = "chiesa") 
                      FILTER(REGEX(?l, "Sicilia", "i"))
                    }

Result

Analysis

La query cerca tutti i beni architettonici o paesaggistici di tipo "chiesa" che si trovano in Sicilia. Per ogni chiesa trovata, la query restituisce l'identificativo del bene (?cp), la sua localizzazione (?l), e l'etichetta dello stile architettonico (?lStile).

8° Question:

What styles are most common in Sicilian churches?

Query:


                    PREFIX dcterms: 

                    SELECT ?nomeStile (COUNT(?stile) AS ?stileCount)
                    WHERE {
                      ?cp a arco:ArchitecturalOrLandscapeHeritage ;
                          dc:type ?o ;
                          a-cd:hasAuthorshipAttribution ?Auth ;
                          dcterms:spatial ?loc .
                      ?loc rdfs:label ?l .
                      ?Auth a-cd:hasCulturalScope ?stile .
                    ?stile rdfs:label ?nomeStile .
                      FILTER(str(?o) = "chiesa") 
                      FILTER(REGEX(?l, "Sicilia", "i"))
                    }
                    GROUP BY ?nomeStile
                    ORDER BY DESC(?stileCount)
                    LIMIT 4

Result

Analysis

La query cerca tutte le chiese in Sicilia e conta il numero di occorrenze di ciascuno stile architettonico tra queste chiese. I risultati sono ordinati in base alla frequenza degli stili, e vengono restituiti solo i primi 4 stili più comuni.

Conclusioni finale - fase SPARQ

Dopo una dettagliata fase di interrogazione dell'ontologia Arco, è emerso un dato sorprendente sulla distribuzione delle chiese nel territorio siciliano rispetto alle altre regioni. Il numero di chiese in Sicilia è significativamente inferiore, suggerendo una potenziale sottorappresentazione nel database. Dal punto di vista architettonico, un'analisi approfondita ha rivelato che oltre 80 chiese siciliane sono classificate come barocche, secondo Wikipedia. Tuttavia, solo 8 di queste chiese barocche sono effettivamente presenti nel database dell'ontologia Arco. Ciò è particolarmente significativo se si considera che l'ontologia comprende più di 350 chiese barocche in totale. Questa discrepanza indica chiaramente una notevole mancanza di dati relativi alle chiese siciliane, in particolare quelle di stile barocco, nel database di Arco. Tale lacuna costituisce un ostacolo alla comprensione completa e accurata del patrimonio architettonico siciliano, evidenziando l'importanza di un'attività di raccolta e inserimento dei dati più approfondita ed estesa. La sezione seguente esplorerà come l'uso dei Large Language Models (LLM) possa essere uno strumento efficace per migliorare il popolamento dell'ontologia, contribuendo così a una rappresentazione più accurata e completa delle chiese siciliane e del loro valore storico e artistico.
